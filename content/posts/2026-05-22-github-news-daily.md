## GitHub 热门项目

### 1. OpenWA

- 开源、自托管的 WhatsApp API Gateway
- 关键词：多会话、Webhook、Dashboard、API Key 认证、n8n 集成
- 适合关注消息系统、自动化和自建通讯基础设施的人

### 2. chrome-devtools-mcp

- 面向 coding agents 的 Chrome DevTools 入口
- 重点能力：性能分析、网络请求检查、截图、控制浏览器自动化
- 很明显是“让 agent 真正会调试浏览器”的方向

### 3. notebooklm-py

- NotebookLM 的非官方 Python API 和 agentic skill
- 覆盖研究导入、问答、导出、音频/视频/图文生成
- 属于知识工作自动化的典型代表

### 4. oh-my-pi

- 终端 AI coding agent，强调 IDE / LSP / DAP 融合
- 主打“更稳的读写改”和更高的工具执行命中率

### 5. multica

- 开源 managed agents 平台
- 强调把 coding agents 变成可分配、可追踪、可协作的“团队成员”

### 结论

今天的技术热度，核心仍是三件事：

- 让 agent 更稳地执行工具

- 让“研究、写作、发布”变成可重复流水线

## 一句话总结

今天的互联网依然在把 AI 从“会说话”推向“能干活”，而真正有价值的项目，开始卷稳定性、可追踪性和发布链路了。
